# CINIVERSE

Discover trending movies, anime, and TV shows all in one place.

## Features

- Browse trending movies, anime, and TV shows
- Powered by TMDB API for up-to-date content
- Responsive design for all devices

## Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** CSS Modules
- **API:** TMDB (The Movie Database)

## Getting Started

```bash
git clone https://github.com/Manojkumar13-ux/CINIVERSE.git
cd CINIVERSE
npm install
npm run dev
```

Create a `.env.local` file with your TMDB API key:

```
NEXT_PUBLIC_TMDB_API_KEY=your_api_key_here
```
